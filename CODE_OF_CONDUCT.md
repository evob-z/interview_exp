# 贡献者公约

## 我们的承诺

为了营造一个开放且热情的环境，作为贡献者和维护者，我们承诺让本项目和社区的参与对所有人都不受骚扰，无论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、教育、社会经济地位、国籍、个人外貌、种族、宗教或性别取向。

## 我们的标准

有助于建设积极环境的行为示例：

- 使用包容、友善的语言
- 尊重不同的观点与经验
- 礼貌地接受建设性批评
- 关注对社区最有益的事
- 对其他成员表现出同理心

不可接受的行为示例：

- 使用带有性别色彩的语言或图像，以及不受欢迎的性骚扰
- 嘲讽、侮辱或贬低性评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息（如住址、邮箱）
- 其他在专业场合中可被合理认定为不当的行为

## 我们的责任

项目维护者有责任明确可接受行为的标准，并对任何不可接受行为采取适当且公正的纠正措施。

维护者有权移除、编辑或拒绝不符合本公约的评论、提交、代码、wiki 编辑、issue 等，并在认为适当时临时或永久封禁任何贡献者。

## 适用范围

本公约适用于项目空间内以及个人代表项目或其社区在公开空间的所有行为。

## 执行

辱骂、骚扰或其他不可接受的行为可通过 [SECURITY.md](SECURITY.md) 中给出的联系方式报告。所有投诉将被审查与调查，并产生维护者认为必要且合适的回应。维护者有义务对事件举报者保密。

## 归属

本公约改编自 [Contributor Covenant 2.1](https://www.contributor-covenant.org/zh-cn/version/2/1/code_of_conduct/)。
